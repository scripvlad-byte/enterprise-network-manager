# Enterprise Network Manager

Enterprise Network Monitoring and Asset Management System built with Python, Flask, MySQL and REST API.

## Overview

Enterprise Network Manager is a web-based platform designed to help IT administrators monitor network devices, manage IT assets, and track network availability from a centralized dashboard.

The project combines networking concepts, database management, automation, and web development into a single solution suitable for small and medium-sized enterprise environments.

## Features

### Network Monitoring

* Device availability checks using ICMP Ping
* Online/Offline status tracking
* Monitoring history logs
* Automated status updates

### Asset Management

* Device inventory management
* Routers, switches, servers, workstations and printers
* Device location tracking
* Device categorization

### Dashboard

* Total devices overview
* Online and offline statistics
* Monitoring activity logs
* Real-time status visualization

### User Management

* Authentication system
* User and administrator roles
* Access control

### REST API

* Device management endpoints
* Monitoring data access
* JSON responses
